# BUG_REPORT_TEMPLATE.md

Bug 反馈模板。

用于帮助开发者或测试者更高效地描述问题，减少“有问题但不知道具体哪里有问题”的低效反馈。

---

## 推荐模板

### 1. 背景
- 哪个页面 / 哪个功能 / 哪个场景

### 2. 复现步骤
1. ...
2. ...
3. ...

### 3. 实际现象
- 现在看到什么
- 从哪一步开始不对

### 4. 预期结果
- 正常应该是什么样

### 5. 补充信息（如有）
- 控制台报错
- 接口返回
- 日志原文
- 截图 / 录像
- 是否最近改过某块相关代码

---

## 一个简单原则

先把现象描述清楚，再给猜测；不要让猜测替代事实本身。
